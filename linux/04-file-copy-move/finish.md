# Nice.

You now know:

- `cp SRC DST` for single files, `cp -r` for directories.
- `SRC/.` copies contents, `SRC` copies the directory itself.
- `mv` renames if source and destination are on the same filesystem.
- `cp -n` (no-clobber) protects against silent overwrites in scripts.
