If `/sbin/nologin` doesn't exist on this image, `/usr/sbin/nologin` or `/bin/false` work just as well — the check accepts any `nologin`- or `false`-family shell. Verify what you actually set with `getent passwd deploy | cut -d: -f7`. If `useradd` complains the user exists, it's already created; move on. If the home dir is missing, re-run with `-m` or create it and `chown deploy:deploy` it.
